---
title: 'Get Product by ID'
api: 'GET /api/Productos/{idProducto}'
description: 'Retrieve a single product by its unique identifier'
---

## Authentication

This endpoint may require authentication depending on your API configuration. Include the JWT token in the Authorization header if required.

```
Authorization: Bearer {token}
```

## Path Parameters

<ParamField path="idProducto" type="integer" required>
  The unique identifier of the product to retrieve
</ParamField>

## Response

Returns a single product object.

<ResponseField name="idProducto" type="integer">
  Unique identifier for the product
</ResponseField>

<ResponseField name="nombre" type="string">
  Product name (max 100 characters)
</ResponseField>

<ResponseField name="precio" type="decimal">
  Product price (decimal with 2 precision)
</ResponseField>

<ResponseField name="img" type="string">
  URL or path to the product image
</ResponseField>

<ResponseField name="stockActual" type="integer">
  Current stock quantity available
</ResponseField>

<ResponseField name="stockMinimo" type="integer">
  Minimum stock threshold before reorder
</ResponseField>

<ResponseField name="idCategoria" type="integer">
  Foreign key reference to the product category
</ResponseField>

<ResponseField name="descripcion" type="string">
  Product description
</ResponseField>

<ResponseField name="Categoria" type="object">
  Related category object with details
  
  <ResponseField name="idCategoria" type="integer">
    Category ID
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    Category name (max 100 characters)
  </ResponseField>
  
  <ResponseField name="descripcion" type="string">
    Category description (max 255 characters)
  </ResponseField>
</ResponseField>

### Success Response (200 OK)

```json
{
  "idProducto": 1,
  "nombre": "Alimento para Perros Premium",
  "precio": 45.99,
  "img": "/images/productos/alimento-perros.jpg",
  "stockActual": 150,
  "stockMinimo": 20,
  "idCategoria": 1,
  "descripcion": "Alimento balanceado premium para perros adultos",
  "Categoria": {
    "idCategoria": 1,
    "nombre": "Alimentos",
    "descripcion": "Alimentos y snacks para mascotas"
  }
}
```

### Error Response (404 Not Found)

```json
"El producto no existee"
```

## Code Example

```bash
curl -X GET https://api.huellitas.com/api/Productos/1 \
  -H "Authorization: Bearer {your_token_here}"
```

```javascript
const productId = 1;
const response = await fetch(`https://api.huellitas.com/api/Productos/${productId}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer ' + token
  }
});

if (response.ok) {
  const producto = await response.json();
  console.log(producto);
} else if (response.status === 404) {
  console.log('Producto no encontrado');
}
```

```python
import requests

product_id = 1
headers = {
    'Authorization': f'Bearer {token}'
}

response = requests.get(f'https://api.huellitas.com/api/Productos/{product_id}', headers=headers)

if response.status_code == 200:
    producto = response.json()
else:
    print('Producto no encontrado')
```
