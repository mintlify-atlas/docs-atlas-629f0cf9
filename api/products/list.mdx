---
title: 'Get All Products'
api: 'GET /api/Productos'
description: 'Retrieve a list of all products in the catalog'
---

## Authentication

This endpoint may require authentication depending on your API configuration. Include the JWT token in the Authorization header if required.

```
Authorization: Bearer {token}
```

## Query Parameters

This endpoint does not accept query parameters. It returns all products.

## Response

Returns an array of product objects.

<ResponseField name="idProducto" type="integer">
  Unique identifier for the product
</ResponseField>

<ResponseField name="nombre" type="string">
  Product name (max 100 characters)
</ResponseField>

<ResponseField name="precio" type="decimal">
  Product price (decimal with 2 precision)
</ResponseField>

<ResponseField name="img" type="string">
  URL or path to the product image
</ResponseField>

<ResponseField name="stockActual" type="integer">
  Current stock quantity available
</ResponseField>

<ResponseField name="stockMinimo" type="integer">
  Minimum stock threshold before reorder
</ResponseField>

<ResponseField name="idCategoria" type="integer">
  Foreign key reference to the product category
</ResponseField>

<ResponseField name="descripcion" type="string">
  Product description
</ResponseField>

<ResponseField name="Categoria" type="object">
  Related category object with details
  
  <ResponseField name="idCategoria" type="integer">
    Category ID
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    Category name (max 100 characters)
  </ResponseField>
  
  <ResponseField name="descripcion" type="string">
    Category description (max 255 characters)
  </ResponseField>
</ResponseField>

## Code Example

```bash
curl -X GET https://api.huellitas.com/api/Productos \
  -H "Authorization: Bearer {your_token_here}"
```

```javascript
const response = await fetch('https://api.huellitas.com/api/Productos', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer ' + token
  }
});

const productos = await response.json();
console.log(productos);
```

```python
import requests

headers = {
    'Authorization': f'Bearer {token}'
}

response = requests.get('https://api.huellitas.com/api/Productos', headers=headers)
productos = response.json()
```
